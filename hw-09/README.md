<h1>Домашнее задание №9</h1>
<h3>Текущее задание:</h3>
<p>Преобразовать консольное приложение каталога книг в библиотеке в веб-приложение с сохранением функциональности.</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Реализовать CRUD операции для сущности книги, а так же отображения авторов, жанров и комментариев.
Остальные операции с сущностями - по желанию/необходимости</li>
<li>Предусмотреть страницу со списком книг, списком авторов, списком жанров,
страницу создания/редактирования книги, страницу просмотра книги со списком комментариев к ней</li>
<li>Удаление сущностей можно сделать на отдельной странице подтверждения удаления, странице редактирования или по кнопке из списка</li>
<li>Для удаления НЕ должен использоваться HTTP-метод GET</li>
<li>Из любого места приложения должна быть возможность вернуться на главную страницу (список книг)</li>
<li>Выбор авторов и жанров при редактировании должен осуществляться из списков (select), ручного ввода id быть не должно </li>
<li>Использовать View на Thymeleaf и классические контроллеры (@Controller)</li>
<li>Протестировать все эндпойнты контроллеров для CRUD операций над книгами с помощью @WebMvcTest и моков сервисов</li>
<li>Локализация интерфейса данным заданием не предусматривается, она строго опциональна</li>
</ul>
