<h1>Домашнее задание №8</h1>
<h3>Текущее задание:</h3>
<p>Перевести приложение каталог книг в библиотеке на использование 
Spring Data MongoDB репозиториев в качестве технологии доступа к данным и MongoDB для их хранения</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Подключить spring-boot-starter-data-mongodb</li>
<li>Переписать все репозитории на Spring Data MongoDB репозитории</li>
<li>Покрыть интеграционными тестами (со встроенной БД) все сервисы</li>
<li>В решении не должно остаться никаких следов других технологий доступа к данным</li>
<li>Работа приложения не должна приводить к наличию данных, находящихся в неконсистентном виде</li>
<li>Учесть отсутствие каскадных операций </li>
<li>Без крайней необходимости из БД не должны тянуться лишние данные</li>
</ul>
