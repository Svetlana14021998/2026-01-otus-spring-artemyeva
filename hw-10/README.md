<h1>Домашнее задание №10</h1>
<h3>Текущее задание:</h3>
<p>Пределать веб-приложение каталога книг в библиотеке на использование архитектуроного стиля REST.</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Для получения данных на страницах приложения использовать JavaScript, fetch api и REST-контроллеры</li>
<li>Минимум: переделать CRUD операции над книгами</li>
<li>URL эндпойнтов должны соответствовать ресурсному стилю REST</li>
<li>Действия над сущностями должны быть выражены исключительно через HTTP-методы. Глаголов в URL быть не должно</li>
<li>Протестировать все эндпойнты REST-контроллеров с помощью @WebMvcTest и моков сервисов</li>
</ul>
