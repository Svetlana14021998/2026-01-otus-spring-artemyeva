<h1>Домашнее задание №4</h1>
<h3>Текущее задание:</h3>
<p>Переписать интерфейс приложения для тестирования студентов с использованием Spring Shell</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Учесть актуальные требования прошлой работы</li>
<li>Подключить Spring Shell, используя стартер</li>
<li>Написать команду/набор команд, позволяющий проводить тестирование 
(можно сделать одну команду, запускающую тестирование)</li>
<li>Учесть, что классы с командами Spring Shell не являются сервисами, они по сути на следующем слое 
(в т.ч. это касается имен классов, пакетов и аннотаций над классами)</li>
<li>Перенести тесты на @SpringBootTest. Компоненты и моки взять из тестового контекста, в остальном,
за исключением интеграции с файловой системой в тесте дао, это должны быть юнит-тесты</li>
<li>Учесть, что Spring Shell в тестах нужно отключить</li>
<li>Локализовывать команды Spring Shell не нужно, а сообщения которые они выводят, нужно</li>
<li>Данная работа не подразумевает использования Spring Application Events</li>
</ul>
