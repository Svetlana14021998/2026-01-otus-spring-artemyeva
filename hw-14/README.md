<h1>Домашнее задание №14</h1>
<h3>Текущее задание:</h3>
<p>На основе Spring Batch разработать процедуру миграции данных из реляционного хранилища в NoSQL или наоборот</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Задание может быть выполнено в отдельном приложении, с сущностями из ДЗ JPA и MongoDB</li>
<li>Не обязательно добавлять процесс миграции в веб-приложение. Приложение может быть оформлено в виде отдельной утилиты.</li>
<li>Используя Spring Batch, следите, чтобы связи сущностей сохранились.</li>
<li>Опционально: Сделать restart задачи с помощью Spring Shell.</li>
</ul>
