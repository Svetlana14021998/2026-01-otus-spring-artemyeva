<h1>Домашнее задание №1</h1>
<h3>Текущее задание:</h3>
<p>Приложение должно загрузить и вывести на экран вопросы с вариантами ответа</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Весь ввод-вывод осуществляется на английском языке, в т.ч. вопросы и варианты ответов</li>
<li>Список вопросов с вариантами ответа хранится в ресурсах приложения, в формате csv</li>
<li>Файл с вопросами читается именно как ресурс, а не как файл</li>
<li>Все зависимости должны быть настроены в IoC контейнере</li>
<li>Контекст описывается XML-файлом</li>
<li>Контекст содержит только компоненты приложения и не содержит стандартных классов и доменных объектов</li>
<li>Имя ресурса с вопросами (csv-файла) необходимо захардкодить строчкой в XML-файле с контекстом</li>
<li>В приложении должна присутствовать объектная модель</li>
<li>Все классы в приложении должны решать строго определённую задачу</li>
<li>Необходимо написать юнит-тест сервиса тестирования</li>
<li>Проверка checkstyle должна проходить успешно</li>
</ul>
