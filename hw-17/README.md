<h1>Домашнее задание №17</h1>
<h3>Текущее задание:</h3>
<p>Обернуть приложение в docker-контейнер</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Обернуть приложение в docker-контейнер. Dockerfile принято располагать в корне репозитория. 
В image должна попадать JAR-приложения. Сборка в контейнере рекомендуется, но не обязательна.</li>
<li>БД в собственный контейнер оборачивать не нужно (если только Вы не используете кастомные плагины)</li>
<li>Настроить связь между контейнерами, с помощью docker-compose</li>
<li>Приложение желательно реализовать с помощью всех Best Practices Docker (логгирование в stdout и т.д.)</li>
</ul>
