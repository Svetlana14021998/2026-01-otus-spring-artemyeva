<p>В CRUD Web-приложение добавить механизм аутентификации</p>
<h4>Требования к реализации:</h4>
<ul>
<li>Добавить в приложение новую сущность - пользователь. Не обязательно
реализовывать методы по созданию пользователей - допустимо добавить пользователей только через БД-скрипты.</li>
<li>В существующее CRUD-приложение добавить механизм Form-based аутентификации.</li>
<li>UserDetailsService реализовать самостоятельно.</li>
<li>Авторизация на всех страницах - для всех аутентифицированных. Форма логина - доступна для всех.</li>
<li>Написать тесты контроллеров, которые проверяют, что все необходимые ресурсы действительно защищены.</li>
</ul>
